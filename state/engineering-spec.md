# OBJECT 001 — Engineering Specification
**Started:** 2026-03-05T22:32:52Z  
**Based on:** product-decision.md + design_final renders

---

## Form Factor
- **Dimensions (target):** 120mm × 80mm × 14mm
- **Weight (target):** <160g
- **Orientation:** Portrait (vertical)
- **Pocket test:** Must fit in standard jeans pocket

## Display
- **Size:** 2.8" IPS LCD
- **Resolution:** 640×480 (4:3 aspect ratio)
- **Refresh rate:** 60Hz
- **Brightness:** ≥400 nits
- **Glass:** Tempered, flush with faceplate

## Processor & Memory
- **SoC:** Rockchip RK3566 (quad-core Cortex-A55, 1.8GHz)
- **RAM:** 1GB LPDDR4
- **Storage:** 16GB eMMC + microSD slot (up to 512GB)
- **GPU:** Mali-G52 (adequate for retro emulation up to PS1/N64)

## Controls
- **D-pad:** Cruciform, single pivot, tactile click
- **Face buttons:** A/B (right side) — OMRON B3F or equivalent
- **Shoulder buttons:** L/R top-mounted
- **System buttons:** Start, Select (small, flush)
- **Switch type:** 100g actuation, positive tactile feedback

## Audio
- **Speakers:** Dual mono, rear-facing, 0.5W each
- **Headphone:** 3.5mm TRS jack (left side)
- **Volume:** Hardware rocker switch

## Power
- **Battery:** 2500mAh Li-Po
- **Charging:** USB-C (5V/2A), bottom edge
- **Runtime:** ~5 hours at moderate brightness
- **Charge time:** ~2 hours

## Shell — Exterior
- **Faceplate:** Brushed aluminum (6061-T6), anodized silver
- **Rear shell:** Brushed aluminum, matching finish
- **Buttons:** Injection-molded ABS, burgundy (#722F37 approx), matte
- **Assembly:** Phillips M2 screws, 6 points
- **Unit numbering:** Laser-engraved rear, "XX/20" format

## Connectivity
- **WiFi:** 802.11 b/g/n (2.4GHz)
- **Bluetooth:** 4.2
- **USB-C:** Charging + data (USB 2.0)
- No cellular, no NFC

## Software
- **OS:** Linux-based, MuOS or similar open-source retro firmware
- **Boot:** Direct to game launcher
- **Emulation targets:** GB, GBC, GBA, NES, SNES, SEGA Genesis, PS1, N64 (partial)

## Compliance Notes
- Not a commercial product — limited to 20 units
- No FCC/CE certification required at this run size
- Internal use / gifting only
