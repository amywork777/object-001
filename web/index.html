⚠️ 6 Compatibility Issues Found & Resolved
1. DPI display used 18+ GPIO pins — no room for buttons → replaced with 2.4" SPI ILI9341
2. DPI display not Pi Zero 2W compatible → SPI ILI9341 proven on Pi Zero 2W
3. TP4056 outputs 3.7V, Pi needs 5V → added MT3608 5V boost converter
4. Battery 65mm wide > 61mm shell interior → 803060 battery (60mm) + shell widened to 75mm
5. No audio components → added MAX98357A I2S DAC + 1W speaker
6. Pi Zero 2W has no GPIO header → added 40-pin header (solder during assembly)

Assembly Sequence (3 days for 20 units)
Day 1 — Prep (2 hrs total)
1. Flash RetroPie to all 20 microSD cards simultaneously (Balena Etcher + USB hub)
2. Solder 40-pin headers onto all 20 Pi Zero 2W boards
3. Solder JST connectors onto TP4056 modules
4. Smoke test each Pi: SD + micro-USB → verify boot
Day 2 — Subassembly (30 min/unit × 20 = 10 hrs, 2 people)
5. Wire MT3608 → Pi 5V pin; set output to 5.15V with multimeter
6. Wire TP4056 battery out → MT3608 input; battery → TP4056
7. Solder SPI display to GPIO per pinout
8. Wire MAX98357A I2S + speaker leads
9. Install 10× tactile buttons in shell, wire to GPIO
10. Full power test before closing shell
11. Close shell with M2 screws; install ABS button caps
Day 3 — Software + QC + Pack (15 min/unit)
12. Boot RetroPie, configure button mapping in RetroArch
13. Load ROM set, verify NES/SNES/GBA all run at full speed
14. QC checklist: display, audio, all buttons, USB-C charge, battery runtime
15. Pack in numbered kraft box (01/20–20/20) with design lineage card
